# NodeMCU sensor

Simple project to upload code to the NodeMCU

Connected hardware

    - BMP085
    - Board led on D4

Board Json

    - http://www.espruino.com/json/ESP8266_4MB.json

Swagger Ui

    - http://localhost:3000
    - http://localhost:3000/swagger.json
